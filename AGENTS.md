# YouTube 字幕翻译扩展

Chrome 扩展，自动翻译 YouTube 视频字幕。

## 技术栈

- Chrome Extension Manifest V3
- TypeScript + esbuild
- 配置通过扩展界面手动设置

## 构建

```bash
npm run build      # 开发构建
npm run build:prod # 生产构建
npm run watch      # 监听模式
```
